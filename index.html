jsDelivr CDN Link Generator

  1. Create a GitHub repository and upload your file.
  2. Enter the URL of your file hosted on GitHub
  3. Copy the generated CDN link using the "Copy URL" button.
  4. You can paste multiple links separated by a space.

About

A simple jsDelivr CDN link generator for GitHub files. This tool allows you to generate a CDN link for your GitHub files hosted on GitHub. The generated link can be used to load your files faster and more efficiently.

Reference: jcubic, jsDelivr
